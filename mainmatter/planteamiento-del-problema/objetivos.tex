\section{Objetivos}
\subsection{Objetivo general}
Implementar un algoritmo clasificador de movimiento v\'alido o inv\'alido a partir del modelo de  detección de los pasos requeridos del movimiento,  con la ayuda del algoritmo de Random Forest Regression y la utilizaci\'on del dispositivo Kinect V2.
\subsection{Objetivos espec\'ificos}
\begin{enumerate}[1.]
\item Definir el movimiento a analizar, estableciendo los pasos requeridos para realizar un movimiento v\'alido.
\item Identificar la distancia mínima y máxima de profundidad entre el Kinect y el usuario para ejecutar el movimiento v\'alido.
\item Grabar los v\'ideos de atletas realizando repeticiones del movimiento v\'alido a partir de la herramienta, Kinect studio.
\item Crear el modelo de detecci\'on de los pasos requeridos de un movimiento v\'alido, aplicando el algoritmo de Random Forest Regression y la etiquetaci\'on de fotogramas de los v\'ideos de repeticiones del movimiento v\'alido.
\item Detectar los m\'argenes de los errores del modelo de detecci\'on de los pasos requeridos de un movimiento v\'alido.
\item Crear un algoritmo clasificador que clasifique la repetici\'on de un movimiento v\'alido, a partir de la detecci\'on de todos los pasos de manera ordenada.
\end{enumerate}
